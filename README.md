# 🏥 Clínica Saúde — Landing Page

Página estática apresentando a Clínica Saúde e seus principais serviços, desenvolvida com HTML5 e CSS3 puro.

## 🔗 Acesso

Basta abrir o arquivo `index.html` diretamente no navegador — sem servidor, sem instalação.

## 📋 Requisitos Atendidos

- ✅ Título na aba do navegador
- ✅ Mínimo de 2 imagens dos serviços
- ✅ Descrição completa dos serviços oferecidos
- ✅ Equipe fictícia com 6 membros (foto, nome e cargo)
- ✅ Uso de H1, H2 e H3 (3 níveis de cabeçalho)
- ✅ Formulário estático com Nome, E-mail, Cidade e Estado

## 🗂️ Estrutura

```
clinica-saude-landing/
└── index.html    ← página completa (HTML + CSS em arquivo único)
```

## 📄 Seções da Página

- **Navbar** — menu de navegação fixo com links para as seções
- **Hero** — apresentação principal com estatísticas da clínica
- **Serviços** — 6 serviços com imagem e descrição
- **Diferenciais** — 6 pontos de destaque da clínica
- **Equipe** — 6 profissionais fictícios com foto e especialidade
- **Formulário** — solicitação de agendamento (estático, sem backend)
- **Footer** — links e informações de contato

## 🛠️ Tecnologias

| Tecnologia | Uso |
|---|---|
| HTML5 | Estrutura da página |
| CSS3 | Estilização (inline no arquivo) |
| Google Fonts | Tipografia (Inter) |
| Unsplash | Imagens dos serviços e equipe |

## 🚀 Como visualizar

1. Clone o repositório:
```bash
git clone https://github.com/henriquesebben/clinica-saude-landing.git
```

2. Abra o arquivo no navegador:
```bash
# Windows
start clinica-saude-landing/index.html

# Mac
open clinica-saude-landing/index.html
```

Ou simplesmente dê **duplo clique** no arquivo `index.html`.

> Não é necessário instalar nada — a página é 100% estática.
